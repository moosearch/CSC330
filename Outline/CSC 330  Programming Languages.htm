Course Overview An introduction to the fundamental concepts in imperative, functional, logic and concurrent programming languages. A language for programming is defined by its syntax and semantics. Syntax defines how a syntactically correct program is constructed; semantics defines how the computation is carried out by such a program. Turing machines, Lambda Calculus and Logical Deduction are three distinctly different models of computation. With concurrency, a common model of computation is that of a (possibly infinite state) transition system, where communication and synchronization are paramount. This course is about the principles underlying the design of these four classes of programming languages, and the idioms in using these languages for solving distinct classes of programming problems.
Topics
  • Introduction
  • History of Programming Languages
  • Syntax and Parsing
  • Names, Types and Type Inference
  • Lambda Calculus
  • Functional Programming with Lists
  • Higher Order Functions and Lazy Evaluation
  • Logic Programming
  • Non-determinism and Backtracking
  • Programming with Lists
  • Concurrent Programming
  • Shared Memory Model and Path Expressions
  • Synchronous and Asynchronous Messaging Model
Course Objectives And Learning Outcomes From this course, you will learn
  • the importance of syntax and semantics of programming languages
  • the structure of a compiler and the techniques of parsing
  • the four major models of computation
  • the different styles of expression supported by each model of computation
  • state-oriented programming based on assignments, control structures and data abstraction
  • expression-oriented programming based on functions, reductions and composition
  • logic-oriented programming based on relations, deductions and non-determinism
  • concurrent programming based on shared memory and message-based synchronization and communication
